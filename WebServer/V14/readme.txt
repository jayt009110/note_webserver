实现动态页面
含有动态数据或资源的页面称为动态页面


动态数据:通过程序执行临时生成的数据为动态数据,资源也一样,
静态数据:事先准备好的固定不变的数据


例如:
实现好的页面,页面上显示某张图片这些都是静态资源,这些文件中的数据是不变的.
而验证码图片就是动态资源,这个图片的数据时通过程序运行临时生成的


实现功能:
在页面上查看所有用户数据:
由于user.dat文件随着用户的操作（注册，修改密码等）数据是不断变化的,因此我们不可能实现准备好的一个静态
页面去包含user.dat文件的所有数据
对此我们需要在用户查看用户列表时通过程序运行，临时生成一个包含当前user.dat文件中所有数据的页面（生成
动态页面）将其响应给客户端

实现过程:
1:在webapps/myweb/index.html上添加一个超链接用于请求生成动态页面的Servlet
  超链接指定的地址为:href = "./showAllUser"
2:在com.webserver.servlet下新建一个类:ShowAllUserServlet并实现service方法.
  service方法用来生成动态页面并设置到response中响应给客户端
3:在ClientHandler的处理请求环节添加分支
